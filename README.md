### What I learned

How to vertically align items with flexbox. 

```html
<body>
  <main>
      <p>Some content</p>
    </div>
  </main>
</body>
```
```css
html, body {
    height: 100%;
}

body {
    display: flex;
    align-items: center;
    justify-content: center;
}
```
